# Cool RxJS Tricks

Presentation files for a talk given at UtahJS Conf 2017.

## Instructions

1. Grab the package.json file:
2. Install the deps:

    ```sh
    npm install
    ```

3. Start the local server:

    ```sh
    npm start
    ```

4. Visit http://localhost:3000 in your browser.
5. Open the development console in your browser.
